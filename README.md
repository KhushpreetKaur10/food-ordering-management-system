# 🍔 Online Food Ordering & Management System

A command-line based food ordering system built using **Python**.  
The application allows customers to browse food items, manage carts and wishlists, place orders, and simulate payments, while administrators can manage products and customers.

---

## 🚀 Features

### 👤 Customer Features
- Customer registration and secure login (via phone or email)
- Browse all available food products
- View products by category
- Search products with search history tracking
- Add and remove items from cart
- Wishlist management
- Place orders and simulate payment
- Cancel orders
- View personal order history

### 🛠 Admin Features
- Admin login system
- Add new food products
- Remove existing products
- View all registered customers
- Add or remove customers
- View complete order history of all users

---

## 📊 Additional Functionality

- Input validation using **Regex**
- Search analytics using **Python Collections (Counter)**
- Persistent storage using **CSV files**
- Order tracking with timestamps
- Secure password input using **getpass**
